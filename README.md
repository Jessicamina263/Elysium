# *About the Project*

ELYSIUM provides a platform to enhance customer engagement and streamline administrative tasks for the cafe and restaurant. The project incorporates robust user and admin systems to ensure a seamless experience for everyone involved.

# *Key Objectives:*
- Deliver a modern, user-friendly interface for customers.  
- Provide comprehensive tools for managing restaurant operations efficiently.  
- Secure all pages with an authentication system.


# *Features:*

# Admin Dashboard
The admin panel includes tools for managing restaurant operations effectively. Key features include:

1. **Product Management**:
   - Add, delete, and edit products (menu items).

2. **Dashboards**:
   - **Contact Dashboard**: Manage customer inquiries and feedback.  
   - **Reservation Dashboard**: Oversee reservations and track availability.  
   - **Employment Dashboard**: Add chefs and manage employee records.  
   - **Admin Management Dashboard**:
     - Add, delete, and update admin accounts.
     - Manage admin authorization levels and permissions.

3. **Authentication**:
   - Access to the admin dashboard is restricted to logged-in users.


# User Pages
The user interface is designed to provide a smooth experience for customers. Key features include:

1. **Homepage**:  
   A welcoming landing page showcasing the restaurant's highlights and unique offerings.

2. **Menu**:  
   Browse the restaurant's curated food and beverage selection.

3. **Reservation**:  
   Book a table with an easy-to-use reservation system.

4. **Contact Information**:  
   View the restaurant's location, operating hours, and contact details.

5. **About Us**:  
   Learn more about ELYSIUM's story, values, and mission.
